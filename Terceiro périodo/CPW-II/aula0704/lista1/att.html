Lista de exercicios 1 - Métodos de String

1. Faça um programa onde o usuário digite um número e o programa diz se é par ou ímpar.



2. Faça um programa onde o usuário digite sua altura e peso e o programa calcula o seu IMC.
3. Faça um programa onde o usuário digite 4 notas e o programa calcula a média.
4. Faça um programa onde o usuário digite 4 números e o programa mostrar o maior deles.
5. Faça um programa que leia 1 palavra e devolva a palavra invertida.
6. Faça um programa que leia uma frase e uma palavra. O programa deve dizer se a palavra existe na frase. O programa deve considerar as diferenças de caixa alta e baixa, por exemplo: "Casa" é diferente de "casa"
7. Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e mostre o maior número.
8. Faça um programa que crie um array com 50 números aleatórios de 0 a 100 e calcule a média dos elementos.
